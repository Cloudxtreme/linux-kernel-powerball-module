
linux-kernel-powerball-module
=============================


Linux Kernel Powerball Module
-----------------------------

A simple Linux Kernel module for getting a Powerball[0] Lottery "quick pick".


PREREQUISITES
-------------

Install Linux Kernel headers and source or a custom Linux Kernel.


INSTALL
-------

`make`
`insmod powerball.ko`


USAGE
-----

`cat /dev/powerball`
12 23 34 45 56 17


[0] http://www.powerball.com/
